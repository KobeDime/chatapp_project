# Kobetrion Chat Application

Welcome to Kobetrion, a dynamic and robust chat application crafted with HTML, CSS, JavaScript, PHP, and MySQL. Kobetrion is designed to deliver a seamless and engaging chat experience, offering real-time messaging paired with a user-friendly interface that works across devices.

## Features

### 1. Real-Time Messaging
- **Instant Communication:** Enjoy uninterrupted and instant chat updates, ensuring smooth conversations with your contacts.

### 2. User Authentication
- **Secure Login & Registration:** Robust user authentication system that keeps your account secure with easy login and registration processes.

### 3. Message History
- **Conversation Archive:** Easily access past conversations with a comprehensive message history feature, allowing you to revisit important chats anytime.

### 4. User Profiles
- **Customizable Profiles:** Personalize your profile with basic information, making it easy to connect with other users.

### 5. Responsive Design
- **Mobile-Friendly Interface:** Kobetrion Chat is designed with responsiveness in mind, ensuring a smooth chat experience on both desktop and mobile devices.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
